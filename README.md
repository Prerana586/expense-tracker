# 💰 Smart Expense Tracker

An AI-powered expense tracker built with Python, Streamlit, and Groq AI.

## Features
- Add expenses using natural language
- AI automatically categorizes and parses expenses
- Beautiful dashboard with charts
- AI-generated financial insights
- Deployed online for free

## Live Demo
[Click here to try the app](https://expense-tracker-ggxbmfce8z5jfhdnakkwhd.streamlit.app/)

## Tech Stack
- Python
- Streamlit
- Groq AI (LLaMA 3)
- SQLite
- Pandas

## How to Run Locally
1. Clone the repo
2. Install dependencies: `pip install -r requirements.txt`
3. Add your `GROQ_API_KEY` to `.env`
4. Run: `streamlit run app.py`
